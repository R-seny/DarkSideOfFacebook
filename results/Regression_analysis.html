Section 1. Predicting PCPTSD score

First, lets visualize the PCPTSD score distribution:

The histogram shape strongly suggests non-normal distribution of the variable under consideration. Moreover, no strong theoretical or other evidence support the idea that we should treat the distances between different test scores to be equal.

So unfortunately, the usage of the ordinary least squares linear regression would be a very doubtful choice, since this model assumes continous dependent variable.

We can perform ordinal logistic regression later (if necessary). ( ?)

Section 2. Predicting WHO score

  • Physical assault from stranger
  • Physical assault from an acquaintance

Indeed, the residuals are not distributed normally in general, even in the model that includes interactions:

This may only indicate the fact that the suggested simple linear regression model is not suitable for our data. Possible solutions include adding more variables in the model (which ones - should be suggested by literature review), switching to the model with lots of dummy-variables, using ordinal regression, using more predictors and principal component analysis to find the underlying variables, explaining most of the variance, and so on.
